Application overview

This dictionary application combines the ease and familiarity of working with a spreadsheet with the power and portability of web pages. It is designed to take dictionary language data contained in a spreadsheet and generate searchable, mobile friendly web-pages from it.

The code is written in the Python programming language and is open source and free to use.

Getting started

Requirements

This program requires Python3 and several Python modules to be installed to interpret and execute the code.

Important files

These are the files you interact with to use the program:

Kovol_lexicon.ods

Is the spreadsheet containing dictionary data. This is the file that users interact with on a daily basis. All data entry and edits are done in this spreadsheet. Both .xlsx and .ods are supported, either Microsoft Excel or LibreOffice Calc are compatible. The file can be renamed and moved to wherever convenient (see lexicon config later). Columns can be rearranged, but the initial layout is recommended. Certain cells have data validation formulae on them that prevent invalid data being entered.

lexicon_config.py

This file should only need to be edited once to set the program up. All options and file locations are set using this file.

lexicon.py

This is the 'executable'.
